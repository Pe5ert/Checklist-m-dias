# Checklist-m-dias

## Visão Geral
Este projeto é um checklist de mídia para cultos que abre já na tela inicial para agilizar o preenchimento.

- Não é necessário ter conta para preencher o formulário.
- Os nomes dos responsáveis aparecem como botões para fácil seleção.
- O histórico é mantido com JavaScript.
- Há campos para título do culto e data do evento.

## Linha do Tempo do Checklist
1. **Abertura inicial**
   - O checklist deve estar visível imediatamente ao abrir a página.
   - Preencher título do culto e data (a data pode ser automatizada para o dia atual).
   - Escolher quem está servindo naquele dia usando botões de nome.

2. **Transmissão**
   - Ligar o PC.
   - Preparar a live (capa, título).
   - Ligar o notebook da iluminação.
   - Ligar o Lumikit Pro 2.
   - Ativar o Zello para comunicação da equipe.
   - Auxiliar o restante da equipe em seus campos.

3. **Projeção**
   - Ligar o telão no quadro de energia.
   - Ligar a controladora do telão.
   - Ligar o PC de projeção.
   - Colocar as músicas em ordem.
   - Ligar as TVs.
   - Testar o vídeo de avisos.

4. **Câmera 1**
   - Pegar a câmera no armário do gabinete pastoral.
   - Ligar a câmera corretamente.
   - Se conectar ao Zello.

5. **Câmera 2**
   - Conferir a bateria da câmera antes do culto.
   - Ligar a câmera 5 minutos antes e testar zoom.
   - Se conectar ao Zello.

6. **Câmera 3**
   - Conferir a bateria da câmera antes do culto.
   - Ajustar o zoom e posição.

7. **Móvel**
   - Pegar a caixa dos transmissores no armário.
   - Pegar o gimbal e câmera.
   - Montar os transmissores e testar.
   - Ligar a câmera e o gimbal.

8. **Pronto para o evento**
   - Tudo deve estar no ponto 10 minutos antes do início do evento.

## Automação e Histórico
- A data pode ser preenchida automaticamente com a data atual do navegador.
- O formulário pode salvar o histórico localmente em JavaScript e permitir exportar para planilhas no futuro.
- O líder pode consultar o histórico depois para conferência.
